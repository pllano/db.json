# Мультиязычность `language`
### По умолчанию

- `id` - integer - id
- `state` - integer - Выкл. = 0 или Вкл. = 1
- `en` - string - English
- `ua` - string - Українська
- `ru` - string - Русский
- `de` - string - Deutsch

### Дополнительно
- `be` - string - Беларускі
- `kk` - string - Қазақша
- `ka` - string - ქართული
- `fr` - string - Français
- `pl` - string - Polski
- `it` - string - Italiano
- `es` - string - Español
- `tr` - string - Türk
- `ar` - string - العربية
- `hi` - string - हिन्दी
- `bn` - string - বাঙালি
- `pt` - string - Portugues
- `jp` - string - 日本語
